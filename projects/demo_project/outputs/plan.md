# 상세 기획안 (PRD): 아이 포토북 (Kids Photo Book)

## 1. 개요 (Overview)

*   **프로젝트명:** 아이 포토북 (Kids Photo Book)
*   **버전:** 1.0
*   **작성일:** 2023년 10월 27일
*   **작성자:** 카리스로 수석 PM
*   **목표:** 아이의 소중한 순간들을 사진, 이야기, 위치, 날씨와 함께 기록하고, 이를 효과적으로 관리하며 공유할 수 있는 모바일 애플리케이션 개발. 투자자들에게 앱의 가치와 시장 잠재력을 명확히 전달하기 위한 상세 기획안.

## 2. 프로젝트 목표 (Project Goals)

*   부모들이 아이의 성장 기록을 쉽고 편리하게 할 수 있도록 돕는다.
*   단순한 사진 갤러리를 넘어, 각 순간에 대한 풍부한 스토리텔링을 가능하게 한다.
*   기록된 추억들을 디지털 및 인쇄 가능한 형태로 보존하고 공유하는 기능을 제공한다.
*   직관적인 사용자 경험(UX)과 안정적인 성능을 통해 사용자 만족도를 극대화한다.

## 3. 주요 기능 (Key Features)

아이 포토북은 다음과 같은 핵심 기능을 제공합니다.

### 3.1. 앨범 생성 (Album Creation)
*   **설명:** 사용자가 여러 장의 사진을 선택하여 하나의 앨범 페이지에 등록할 수 있습니다. 각 앨범은 특정 이벤트나 시점을 중심으로 구성됩니다.
*   **세부 사항:**
    *   사용자는 기기 갤러리에서 다중 이미지 선택을 통해 앨범을 생성할 수 있습니다.
    *   생성된 앨범은 메인 화면의 목록에 추가됩니다.

### 3.2. 상세 기록 (Detailed Record)
*   **설명:** 앨범에 등록된 사진과 함께 다양한 정보를 기록하여 해당 순간의 맥락을 풍부하게 합니다.
*   **세부 사항:**
    *   **날짜:** 앨범 생성 시 자동으로 현재 날짜가 기록되며, 사용자가 수동으로 수정할 수 있습니다.
    *   **촬영 위치 (GPS 자동 감지):** 사진 촬영 시의 GPS 정보를 자동으로 감지하여 기록합니다. 사용자는 위치 정보의 자동 감지 여부를 설정할 수 있으며, 필요 시 수동으로 위치를 입력하거나 수정할 수 있습니다.
    *   **날씨:** 촬영 날짜 및 위치 정보를 기반으로 해당 시점의 날씨 정보를 자동으로 기록합니다. (API 연동 필요)
    *   **전체 이야기 작성:** 해당 앨범 전체에 대한 설명을 자유롭게 작성할 수 있는 텍스트 입력 필드를 제공합니다.

### 3.3. 개별 캡션 (Individual Caption)
*   **설명:** 앨범 내의 각 사진마다 독립적인 캡션(caption) 또는 설명을 작성할 수 있는 기능을 제공합니다.
*   **세부 사항:**
    *   각 사진을 탭하면 편집 모드에서 해당 사진에 대한 개별 캡션을 추가/수정할 수 있습니다.
    *   텍스트 입력 길이에 제한을 두어 간결한 설명을 유도합니다.

### 3.4. 날짜별 목록 (Date-based List)
*   **설명:** 생성된 모든 앨범을 월별로 그룹화하여 목록 형태로 제공합니다.
*   **세부 사항:**
    *   메인 화면에서 앨범 목록을 스크롤하면 월별 구분선과 함께 앨범이 정렬됩니다.
    *   최신 앨범이 상단에 위치하도록 정렬됩니다.

### 3.5. PDF 내보내기 (PDF Export)
*   **설명:** 사용자가 선택한 앨범들을 소책자(booklet) 형태의 PDF 파일로 생성하고 공유할 수 있습니다.
*   **세부 사항:**
    *   앨범 선택 화면에서 내보내고 싶은 앨범들을 다중 선택합니다.
    *   선택된 앨범들의 사진과 기록된 정보(날짜, 위치, 날씨, 전체 이야기, 개별 캡션)를 포함하여 PDF를 생성합니다.
    *   생성된 PDF는 이메일, 메신저 등 다양한 채널을 통해 공유할 수 있습니다.
    *   인쇄 친화적인 레이아웃으로 디자인됩니다.

### 3.6. 풀스크린 사진 보기 (Fullscreen Photo View)
*   **설명:** 앨범 내의 특정 사진을 탭하면 해당 사진을 전체 화면으로 감상할 수 있습니다.
*   **세부 사항:**
    *   전체 화면 보기 시, 스와이프 제스처를 통해 동일 앨범 내 다른 사진으로 이동할 수 있습니다.
    *   사진 하단 또는 오버레이 형태로 개별 캡션이 표시됩니다.

## 4. 기술 스택 (Technical Stack)

본 프로젝트는 다음과 같은 기술 스택을 활용하여 개발되었습니다.

*   **Framework:** Expo (React Native) + TypeScript
    *   크로스 플랫폼 개발 및 개발 생산성 향상
    *   타입 안정성 및 코드 품질 보장
*   **Navigation:** React Navigation v7
    *   앱 내 화면 전환 및 내비게이션 관리
*   **Storage:** AsyncStorage
    *   로컬 데이터 저장 및 관리
*   **Image Picker:** expo-image-picker
    *   기기 갤러리에서 이미지 선택 기능 구현
*   **Location:** expo-location
    *   GPS를 이용한 위치 정보 자동 감지 및 기록
*   **PDF:** expo-print + expo-sharing
    *   PDF 문서 생성 및 공유 기능 구현
